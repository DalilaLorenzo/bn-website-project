# Barnes & Noble Website Analysis – HTML/CSS Project

This project is a multi-page website created as part of a web development assignment. It includes an overview of the Barnes & Noble company, a critique of their website, and proposed solutions to improve user experience, performance, and design.

## Pages Included
- index.html – Company overview and introduction  
- webcritique.html – Strengths and weaknesses of the Barnes & Noble website  
- solutions.html – Recommended improvements for UX, performance, and security  

## Technologies Used
- HTML  
- CSS  
- Embedded images and video  
- Basic layout and navigation structure  

## Project Structure
- index.html  
- webcritique.html  
- solutions.html  
- style.css  
- Images/ folder  

## Purpose
The purpose of this project is to practice building structured web pages using HTML and CSS, while also analyzing a real-world website from a user experience perspective.
